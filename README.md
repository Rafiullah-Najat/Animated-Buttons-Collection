# Animated-Buttons-Collection

 This project is a collection of **modern animated buttons** built using pure HTML and CSS.
 ---

## 🚀 About the Project

This is the **first part** of the Animated Buttons series.  
More creative and advanced button designs will be added in future updates.

The goal of this project is to:
- Practice CSS animations
- Build reusable UI components
- Create visually attractive button effects

---

## ✨ Features

- Smooth hover animations
- Neon glow effects
- Pure CSS (no JavaScript)
- Lightweight and responsive design
- Easy to customize colors and styles
